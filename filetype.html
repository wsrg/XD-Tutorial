Choosing a file type

Adobe XD allows you to design for various digital outputs. To make this easy, it offers three standard artboard sizes and one custom option. The three standard artboard sizes are:

  • iPhone
  • iPad
  • Web

To change the artboard dimensions for each of these options, select the down arrow to the right of each option. If your desired output does not match either of these three, you may always select Custom Size to manually create your artboard.

To create a web document, select Web 1920.
