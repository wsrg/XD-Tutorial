Tools

Adobe XD offers various basic design tools to help you create your prototype.

  • Select: Use the selection tool to select objects on your canvas. Selected objects will be displayed as surrounded by a colored bounding box.

  • Square: Draw a square or rectangle object on the canvas.

  • Circle: Use this shape tool to draw a circle.

  • Triangle: Create triangle shapes with this tool.

  • Line: Use the line tool to raw lines or linear objects on your canvas.

  • Pen: Use the pen tool to draw custom shapes on your canvas.

  • Text: Add text to your design using the text tool. Text appearance can be modified using the edit console.

  • Artboard: Create multi-page prototypes by adding artboards to your canva.

  • Magnifier: Click or hold to select an area of the canvas to zoom in.

  • Assets: Manage components which you have created within the document in the Assets menu. To learn more about components, go to Making Components.

  • Layers: Each object added to an artboard creates a new layer on the canvas. Use the layers toolbox to view and manage your layer.

  • Plug-ins: View, manage, and use installed Plug-ins. You can also download more Plug-ins using the "add" icon in the Plug-ins toolbox.
