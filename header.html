Header


Adobe XD’s header allows you to navigate the application’s different functions.

  • A sandwich icon opens the File menu. The File menu allows you to perform standard actions, such as open a file, save a document, get UI Kits, as well as import and export files.

  • The home button reopens XD’s home page, seen in Figure 1.1.

  • The most important feature within Adobe XD’s header is the ability to switch between XD’s three main work modes: Design, Prototype, and Share. Each of these offers its own array of capabilities that to help you create your prototype.

  • The title of the document is located in the center of the header. To change the title, click on the current title and enter the new name in the pop-up window. Save the new name when you’re done.

  • The add user icon allows you to invite other contributors into the document. This tool allows those who you have invited to download the document from the cloud and make changes to the document on their own device.

  • The mobile icon lets you preview the document on a mobile device. To do this, you will need to download the Adobe XD mobile application.

  • Use the Play button to preview the document on your computer screen.

  • Change the view of your document by using the manual zoom on the top-right of the header.
